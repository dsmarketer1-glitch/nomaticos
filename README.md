# Nomatic OS — Agency Operating System

> AI-powered personal agency operating system for managing clients, tasks, and daily workflow.

## Tech Stack
- **Frontend:** React 18 + Vite + TailwindCSS v3
- **Backend:** Node.js + Express.js
- **Database:** SQLite (via better-sqlite3)
- **AI:** Claude API for monthly plan generation

## Setup

```bash
# Install all dependencies
npm install
cd client && npm install && cd ..

# Run both frontend and backend together
npm run dev

# Frontend runs on: http://localhost:5173
# Backend API runs on: http://localhost:3001
```

## Features
- Client management with onboarding checklists
- Task management (Table + Kanban views)
- AI-generated monthly marketing plans
- End of Day (EOD) reporting
- Dark/Light mode
- Fully local — no cloud dependencies
